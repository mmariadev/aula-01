Paragráfo

A tag p no HTML representa um parágrafo, servindo para organizar textos em blocos estruturados. Ela agrupa frases, imagens ou outros conteúdos relacionados, criando automaticamente um espaçamento vertical antes e depois do texto, facilitando a leitura e melhorando a semântica do documento.

Tag de abreviações e acronimos

A tag abbr é utilizada para definir uma abreviação ou um acrônimo (como "HTML", "CSS", "Dr.", "SP"). Ela é um elemento semântico que ajuda mecanismos de busca, leitores de tela e navegadores a entenderem que aquele texto é uma forma abreviada de uma expressão maior.

Tag de referência "cite"

A tag cite no HTML é um elemento semântico usado para representar a referência de uma obra criativa, como o título de um livro, filme, música, ou o nome do autor de uma citação. Ela indica que o texto contido é uma citação ou referência, geralmente renderizada em itálico pelos navegadores.

"del" tag de exclusão de texto

A tag del no HTML é utilizada para marcar texto que foi excluído ou removido de um documento, exibindo-o com um risco horizontal (tachado) por padrão. Ela é fundamental para indicar alterações, revisões, controle de versão ou preços antigos em promoções, geralmente acompanhada da tag ins para mostrar o conteúdo que o substituiu.

"dfn" tag de definição

A tag dfn(definition element) no HTML serve para indicar, de forma semântica, o termo que está sendo definido dentro de um contexto, como em um glossário, parágrafo explicativo ou documentação. Ela marca a primeira vez que um conceito é introduzido e explicado, ajudando motores de busca e leitores de tela a identificar definições.

"em" Tag para enfatizar determinado conteúdo.

A tag "em" no HTML serve para indicar ênfase semântica em um trecho de texto, alterando sutilmente o significado da frase. Por padrão, os navegadores a exibem em itálico, mas seu uso principal é informar a leitores de tela e mecanismos de busca que o conteúdo é importante.
